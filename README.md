# 💤 LazyVim

A starter template for [LazyVim](https://github.com/LazyVim/LazyVim).
Refer to the [documentation](https://lazyvim.github.io/installation) to get started.

## 配置项

```sh
./lua
├── config
│   ├── autocmds.lua # 行内提示优化
│   ├── keymaps.lua # 键位映射
│   ├── lazy.lua # 插件管理
│   └── options.lua # 全局设置
└── plugins
    ├── catppuccin.lua # 颜色主题
    ├── opencode.lua # AI Agent
    ├── tiny-inline-diagnostic.lua # 行内提示优化
    └── wakatime.lua # 时间统计
```
